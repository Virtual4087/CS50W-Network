# Syncly

**Syncly is a reddit like social media platform where you can share your opinions.**

 # Setup Guide
- Download the source code
- Install Django
- Install Sqlite3
- Run python manage.py makemigrations
- Run python manage.py migrate
- Run python manage.py runserver

## Techstack used:
- Backend
  - Python
    - Django

- Frontend
  - CSS
    - Bootstrap
  - Javascript

- Database
   - Sqlite3
